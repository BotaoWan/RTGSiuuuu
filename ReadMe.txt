make sure that jq 64-bit is installed and set properly


use cmd and trade.bat to quickly add , delete and change traders

add trader: trade.bat add TRADER_NAME
delete trader: trade.bat del TRADER_NAME
change trader name: trade.bat change TRADER_NAME_BEFORE TRADER_NAME_AFTER

Change to dataset4: trade.bat set data 4
Change name of match to MatchVS: trade.bat set match MatchVS
Change speed of match to 2(only integers allowed): trade.bat set speed 2

check the dataset number using: trade.bat get data
check the match name using: trade.bat get match
check the speed of match using: trade.bat get speed
