make sure that jq-win64 is installed and set properly

first use cmd to go to the current directory

then use cmd and trade.bat to quickly add , delete and change traders

EXAMPLES:

add trader with tradername siuuu: trade.bat add siuuu (recommended to first add file "siuuu.py" to the folder "Trader")
delete trader siuuu: trade.bat del siuuu
change trader name from siuuu to siuuu_original: trade.bat change siuuu siuuu_original


Change to dataset4: trade.bat set data 4
Change name of match to MatchVS: trade.bat set match MatchVS

check the dataset number using: trade.bat get data
check the match name using: trade.bat get match
